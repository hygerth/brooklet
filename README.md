# brooklet

```
Usage: brooklet [OPTIONS]
Important notes: brooklet expects MongoDB to run locally on port 27017
Version: 2016.01.15
Options:
-h, -help       Print this help text and exit
-v, -version    Print program version and exit
-duration       Specify duration for cached content, default 5
-timeunit       Specify time unit for cache duration, default minutes
-port           Specify server port, default 9876
```
